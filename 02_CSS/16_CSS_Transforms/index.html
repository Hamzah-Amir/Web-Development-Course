CSS Transform Methods Explained

1. Translate

transform: translate(x, y) - Moves an element from its current position. The first value moves horizontally (positive = right, negative = left), the second moves vertically (positive = down, negative = up).

2. Rotate

transform: rotate(angle) - Rotates an element around its center point. Positive values rotate clockwise, negative values rotate counter-clockwise. Angle is specified in degrees (deg).

3. Scale

transform: scale(value) - Changes the size of an element. Values greater than 1 make it larger, values less than 1 make it smaller. You can also use scaleX() and scaleY() for horizontal/vertical scaling.

4. Skew

transform: skew(x-angle, y-angle) - Tilts an element by the specified angles. Creates a parallelogram effect. You can use skewX() and skewY() for single-axis skewing.

5. Multiple Transforms

You can combine multiple transforms in one declaration: transform: translate() rotate() scale(). The order matters - transforms are applied from right to left.

Transform Origin

transform-origin - Controls the point around which transforms happen. Default is center, but you can set it to top, bottom, left, right, or specific coordinates.
