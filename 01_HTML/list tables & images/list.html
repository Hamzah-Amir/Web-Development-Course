Hyper Text Markup Language - HTML
Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content.

JavaScript (JS) is a programming language used to create dynamic content for websites. It can be used to make web pages more interactive, such as by adding animated graphics, drop-down menus, and interactive maps.
